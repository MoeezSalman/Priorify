# Priorify

An intelligent project management support system that analyzes user feedback using AI-powered sentiment analysis and automatically prioritizes feature requests to support data-driven sprint planning.

🚀 Overview

This system enables Project Managers to:

Analyze user feedback using AI sentiment analysis

Automatically prioritize feature requests

Manage Product and Sprint backlogs efficiently

Visualize feedback trends through interactive dashboards

Export reports with insights and visual graphs

The platform ensures that product decisions are aligned with real user needs by combining sentiment scores and frequency of mentions.

👤 User Role

Project Manager

The system is designed specifically for Project Managers to manage feedback, backlog items, and sprint priorities effectively.

✨ Key Features
🔐 Authentication

Secure login using credentials

Logout functionality to protect account access

📊 Dashboard & Insights

View total number of feedback items

Real-time automatic feedback count updates

Weekly and monthly sentiment trend analysis

Pie chart for sentiment distribution (Positive / Negative / Neutral)

Bar chart for High / Medium / Low priority distribution

Horizontal bar chart of most frequent negative words

🤖 AI-Powered Sentiment Analysis

Automatic sentiment calculation for all feedback

Sentiment score assigned to each feature request

Urgent feature highlighting

Priority suggestion based on:

Sentiment score

Number of mentions

📋 Backlog Management
Product Backlog

All unresolved feedback automatically added

Resolved requirements automatically removed

Filter features by sentiment (Positive, Negative, Neutral)

Clear filter indicators

Manual priority override option

Sprint Backlog

Automatically selects top 3 high-priority features

Moves selected features from Product Backlog to Sprint Backlog

Supports data-driven sprint planning

📑 Feature Request Table

Tabular view of all feature requests

Displays:

Feature Name

Sentiment Score

Priority

Export table for stakeholders

Export prioritized table as PDF

PDF includes visual graphs for requirements engineers

📈 Data Visualization

The system includes:

🟢 Pie Chart – Sentiment Distribution

📊 Bar Chart – Priority Distribution

📉 Sentiment Trend Graph (Weekly/Monthly)

📌 Horizontal Bar Chart – Most Frequent Negative Keywords

These visualizations help stakeholders quickly understand feedback trends and workload distribution.


📤 Export Capabilities

Export prioritized features as PDF

PDF includes:

Feature table

Priority distribution charts
